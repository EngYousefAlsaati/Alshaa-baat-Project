## 🛠️ التقنيات | Stack

- **HTML5** — semantic markup, schema.org structured data
- **CSS3** — custom properties, logical properties, container queries-ready
- **Vanilla JavaScript** — IntersectionObserver for scroll reveals
- **Fonts:** Tajawal (Arabic) + IBM Plex Sans (Latin) via Google Fonts

No build step. No dependencies. Ready to deploy.

## 🚀 التشغيل | How to Run

### محلياً | Local
```bash
git clone https://github.com/YOUR-USERNAME/alshaa-maintenance-supply.git
cd alshaa-maintenance-supply
# Open index.html in browser, or:
python3 -m http.server 8000
# Then open http://localhost:8000
```

### GitHub Pages
1. Push to a GitHub repo
2. Repo Settings → Pages → Source: `main` branch / `root`
3. Wait for the deployment URL
